# 靈晶祝福 (Crystal Blessing) 網站設計與操作說明手冊

## 1. 網站概述
「靈晶祝福」是一個結合東方命理、五行學說與現代 AI 技術的客製化水晶能量推薦電商網站。使用者可以透過填寫個人資訊與祈願需求，獲得專屬的流年運勢分析、水晶手鍊配方建議，以及神明加持推薦，並直接在線上將專屬的推薦水晶加入購物車進行結帳。

## 2. 技術架構
* **前端 (Frontend)**：HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
* **後端 (Backend)**：Node.js + Express
* **資料庫 (Database)**：Google Sheets API (作為訂單、價格與設定的輕量級資料庫)
* **AI 引擎**：Google Gemini API (主要) + OpenAI GPT-4o-mini (備用/Fallback)
* **通知系統**：Nodemailer (透過 SMTP 發送訂單確認信)

## 3. 核心功能與頁面介紹

### 📍 1. 祈願首頁 (`index.html`)
* **動態視覺**：首頁包含一個 3D 旋轉的「水晶星球系統」，使用者可以用滑鼠互動，點擊星球可查看對應水晶資訊。背景有星空特效 (`stars.js`)。
* **祈願表單**：使用者需填寫基本資料（姓名、性別、出生年月日）、選填資料（出生時間、宗教信仰、手圍、聯絡方式），以及下拉選擇或自填「祈願項目」（如感情、事業、健康等）。
* **能量報告生成**：送出表單後，系統會呼叫 AI 生成報告。結果區塊分為三個標籤頁（且在手機版會吸附置頂）：
  1. **🔯 能量分析**：基於八字五行的流年與命格分析。
  2. **💎 水晶推薦**：AI 會根據五行缺漏與祈願，設計 2~4 條專屬「文謅謅名稱」的水晶手鍊配方，並附帶「🛒 加入購物車」按鈕。
  3. **🙏 神明加持**：根據使用者的宗教信仰與命格，推薦適合祈求的神明或守護天使。

### 📍 2. 水晶百科 (`crystals.html`)
* 採用網格排版展示所有可用的水晶種類。
* 點擊任一水晶會跳出「說明視窗 (Modal)」，顯示水晶的詳細五行屬性、對應脈輪、核心功效。
* 視窗內附帶「🛒 加入購物車」按鈕，讓使用者也可以直接單買特定的水晶手鍊。

### 📍 3. 購物車與結帳 (`cart.js`)
* **側邊欄設計**：點擊右上角購物車圖示會從右側滑出購物車面板。
* **動態計算**：自動計算商品小計、手續費、預計運費與總計。
* **結帳流程**：
  1. 點擊「複製訂單/結帳」會跳出結帳表單。
  2. 使用者填寫收件資訊（寄送方式、付款方式、收件地址、Email、手機）。
  3. 送出後，系統會將訂單更新至 Google Sheets，並發送 HTML 格式的確認信到使用者的 Email。

### 📍 4. 查詢訂單
* 位於上方導覽列的「🔍 查詢訂單」。
* 使用者輸入結帳時留下的「手機號碼」，系統會去 Google Sheets 撈取歷史訂單，並顯示訂單時間、購買明細與狀態。

### 📍 5. 管理後台 (`admin.html` / `admin.js`)
* **資料來源**：直接讀取與寫入 Google Sheets。
* **功能**：
  * **商品定價管理**：同步 Google Sheets 的 `Prices` 工作表，可以在此調整水晶的單價與名稱。
  * **系統設定**：同步 `Settings` 工作表，管理如「運費」、「手續費」、「每日 AI 限制次數 (MAX_AI_CALLS_PER_DAY)」等全站變數。
  * **訂單匯出**：可以將 Google Sheets 中的訂單資料快速匯出查看。

## 4. 系統與後端整合細節

### 🔄 AI 容錯機制
* 系統主要依賴 `GoogleGenerativeAI (Gemini-2.5-flash)`。
* 若 Gemini 發生配額不足或連線錯誤，系統會自動 fallback 至 `OpenAI (GPT-4o-mini)` 確保服務不中斷。
* 若兩者皆失敗，系統有內建的 `generateMockResult` 假資料機制，避免前端崩潰。

### 📊 Google Sheets 整合 (資料庫)
網站依賴一個 Google Sheet，內部包含三個主要工作表：
1. **Orders**：記錄使用者的祈願資料、AI 分析結果、以及後續結帳的物流與金額明細。
2. **Prices**：記錄水晶代號、名稱與價格，前端購物車會動態從此處撈取價格。
3. **Settings**：存放全域變數設定（如運費、免運門檻等）。

### 📧 Email 通知系統
* 採用 `nodemailer` 模組，透過 Gmail SMTP (App Password) 發信。
* 使用者結帳完成後，會收到一封排版精美的 HTML 訂單確認信，內含購買的水晶手鍊明細、數量、金額與運送方式。

## 5. 網站視覺與設計美學
* **色彩配置**：以深空紫 (`#0a0414`) 為底色，搭配神秘紫 (`#7c3aed`) 與星芒金 (`#f59e0b`)，營造出高質感、靈性、神秘的氛圍。
* **排版特效**：
  * **Glassmorphism (毛玻璃效應)**：卡片與導覽列廣泛運用半透明背景與模糊濾鏡 (`backdrop-filter: blur`)，呈現現代高級感。
  * **微動畫 (Micro-animations)**：按鈕 Hover 時有浮動與光暈效果，購物車徽章有跳動動畫，AI 打字機效果 (`typewrite`) 增加互動儀式感。
* **RWD 響應式設計**：
  * 完美支援手機版與桌機版。
  * 手機版採用「標籤列置頂 (Sticky Tabs)」，方便使用者在長篇的能量報告中快速切換分析、水晶與神明頁籤。
  * 水晶推薦卡片在手機版自動轉為垂直排版，確保文字不斷行且按鈕滿版易於點選。
